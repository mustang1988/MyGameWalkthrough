小段剧情后, 前往加雷利亚要塞, 继续主线剧情, 回复装置处保存游戏, 剧情后Boss战

*陷阱师杰诺*, *破坏兽雷欧尼达斯*

!> 一般胜利条件是将其中一人HP削减4万, 40回合内两人HP削减4万就能提前结束战斗并*AP+3*

两人的攻击力都是中上伤害, 要注意杰诺会丢三根罪恶掷枪(记得调查)

记得一定要用范围战技清掉, 否则爆炸带有随机异常效果

要是中石化或混乱就一定失败, 雷欧尼达斯则是攻击力高

且带有昏厥, 尤其S战技伤害, 建议以黎恩与菲组成攻击队伍, 负责输出

艾略特与托瓦尔为辅, 至少其中一人装备回路强音之力, 可以少放一次黎恩的激励

当全员都有STR+50%的状态时, 由黎恩发动增幅强化, 记得在还没决定给予猛攻时

只能集中攻击BOSS其中一人, 以防两人都同时施放S战技导致团灭

剧情后接着骑神战

*龙人・刀*, *龙人・枪*

有两台龙人, 集中攻击一台即可, 打倒其中一台时, 会抢夺对方的剑

此时转变为对付龙人‧刀跟龙人‧枪, 并开启同伴模式

此刻绊最高的同伴会自动参战, 当然可以替换, 每个同伴都可以发挥效果不同的EX魔法, 

战斗结束后该角色和黎恩的好感+100

龙人‧剑的弱点：
- 一般状态：身体, 
- 蓄力状态：头部, 

龙人‧枪的弱点：
- 一般状态：头部, 
- 蓄力状态：身体, 

当前战斗笔记: 52

战斗结束后剧情, 剧情结束后, 可以在要塞自由活动
